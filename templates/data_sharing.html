Data Sharing Information

Community-Based Research

The Educational Ransomware Simulation tool is part of a community-driven effort to better understand and combat ransomware threats. By sharing anonymized data about your simulations, you contribute to this important research.

What Data is Shared

When you use this tool, the following data may be shared with the security community:

  • File hash values
  • Encryption patterns and techniques identified
  • Technical indicators that could help identify similar ransomware
  • Statistical information about encryption/decryption performance

What Is NOT Shared

We do not share the following types of information:

  • Original file contents
  • Personal information
  • IP addresses or location data
  • Any information that could identify individual users

Benefits of Sharing

By participating in data sharing, you contribute to:

  • Development of better ransomware detection tools
  • Creation of more effective recovery strategies
  • Building a comprehensive database of ransomware techniques
  • Advancing cybersecurity education and awareness

Control Over Your Data

You maintain control over your data sharing preferences. If you wish to opt out of data sharing, please contact the administrator of this system.

Security Commitments

All shared data is transmitted securely and stored in accordance with industry best practices. We implement appropriate technical and organizational measures to protect the data that is collected and shared.
